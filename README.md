# 🌐 3D Portfolio

This is an interactive 3D portfolio built with **React Three Fiber**, **@react-three/drei**, and **Framer Motion**. It showcases 3D models, smooth animations, and responsive design to give a modern and engaging experience.

## 🚀 Tech Stack

- [React](https://reactjs.org/)
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/)
- [@react-three/drei](https://github.com/pmndrs/drei)
- [Framer Motion](https://www.framer.com/motion/)
- [Three.js](https://threejs.org/)
- [GLTF models](https://github.khronos.org/glTF-Validator/)
